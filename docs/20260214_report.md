## 2. 개선 적용 상태(요약)

- exploit 8개 관련 P0/P1/P2 권고안은 모두 반영됨.
  - verdict 표준화(3값) / “two final files” 제거 / exploit 공통 include(`_exploit-shared-core`) 도입
  - 빈 queue evidence 허용(validator 정렬)
  - Playwright 병렬 충돌 완화(카테고리별 MCP 매핑 분산)
  - prompt lint 추가(`npm run prompt-lint`)
  - exploit 결과 집계에서 `BLOCKED_BY_SECURITY` 별도 표시

- vuln 8개 관련 권고안은 대다수 반영되었고, 일부는 warning 레벨로만 적용됨.
  - 상세 적용 현황은 `docs/20260214_vuln_report.md`의 “개선 적용 현황” 참고.
