**Path Recovery Strategy (MANDATORY):**
If a file reading tool (e.g., `open_file` or `view_file`) fails with "No such file or directory":
1. **Verify CWD:** Check your current working directory using `pwd` via your shell tool (e.g., `bash` or `run_command`).
2. **Try Repo Anchor:** Prepend `{{REPO_PATH}}/` to your relative path and retry.
3. **Search for Basename:** If both fail, search for the file by its name to find its actual location:
   - Tool: Use your search or shell tool (e.g., `bash`, `run_command`, `rg`, or `search_file`) to find the file.
   - Example (bash): `find . -name "<basename>"` or `rg --files -g '*<basename>*'`.
4. **List Parent:** Use your directory listing tool (e.g., `ls` or `list_dir`) on the suspected parent directory to see available files.
