### 6.2 Playwright Browser Automation & Network Analysis (MANDATORY)

**You MUST use {{MCP_SERVER}} (Playwright) or curl (via bash) to discover runtime behavior that static analysis cannot reveal.**

**Required Tasks:**

1. **Client-Side Route Discovery (Playwright)**
   - Navigate to the application root
   - Take snapshot to extract all visible links and navigation elements
   - Click through menus to discover SPA routes (React Router, Vue Router, etc.)
   - Document all discovered routes with their access requirements

2. **Dynamic API Endpoint Discovery**
   - Monitor network requests while navigating the application
   - Capture AJAX calls, fetch requests, and WebSocket connections
   - Document API endpoints not visible in source code (minified bundles, lazy-loaded modules)

3. **Authentication Flow Mapping**
   - Navigate through login process
   - Observe session token generation and storage (cookies, localStorage)
   - Test session persistence across page reloads
   - Identify multi-step authentication (2FA, OAuth flows)

4. **Role-Based UI Discovery**
   - If credentials are available, log in as different user roles
   - Document role-specific menus, buttons, and features
   - Identify hidden admin interfaces or privileged functionality

5. **Interactive Element Inspection**
   - Use browser_evaluate to inspect DOM for hidden forms
   - Check for client-side validation logic
   - Identify JavaScript-based routing or state management

   - **MANDATORY**: Periodically use `{{MCP_SERVER}}__browser_network_requests` to inspect traffic captured by the browser.
   - Use `curl` via the `bash` tool for direct protocol-level probing when browser behavior is insufficient.
   - ‚ö†Ô∏è **BASH TOOL USAGE**: Ensure the `command` argument is a **plain shell command string**. Do NOT wrap it in JSON.
     - ‚úÖ Correct: `{"command": "curl -i http://..."}`
     - ‚ùå Incorrect: `{"command": "{\"command\": \"curl ...\"}"}`



**Example Playwright & curl Workflow:**
```
1. {{MCP_SERVER}}__browser_navigate ‚Üí {{WEB_URL}}
```
